# What Happens When I Access a Site

## Basic flow
1. I type a URL into the browser
2. The browser checks if the address is cached
3. The system performs DNS resolution
4. The IP address is returned
5. A connection is established to the server
6. If HTTPS is used, TLS negotiation happens
7. The browser sends an HTTP request
8. The server returns an HTTP response
9. The browser renders the page

## Why this matters
This flow connects networking, DNS, web protocols, and basic security concepts.
