# How DNS Works

## What is DNS?
DNS stands for Domain Name System. It translates human-readable domain names into IP addresses.

## Why DNS is important
Without DNS, users would need to remember IP addresses instead of domain names.

## Basic resolution flow
1. The user types a domain name in the browser
2. The device checks local cache
3. A DNS query may be sent to a recursive resolver
4. The resolver looks for the correct DNS records
5. The IP address is returned
6. The browser connects to the destination server

## Common DNS record types
- A
- AAAA
- CNAME
- MX

## Why DNS matters in cybersecurity
DNS is important for visibility, filtering, troubleshooting, and threat detection.
