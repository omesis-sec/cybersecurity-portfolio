# Windows Event Viewer Basics

## What is Event Viewer?
Event Viewer is a Windows tool used to view system, security, and application logs.

## Why it matters
It helps identify errors, warnings, failed logins, and system events.

## Main log categories
- Application
- Security
- System

## Common use cases
- checking system errors
- reviewing login-related events
- troubleshooting service failures

## Why this matters in cybersecurity
Logs are essential for monitoring, troubleshooting, and basic investigation.
